# MongoDB Practice

A practice project built with Node.js, Express, MongoDB, and Mongoose to learn schema creation, validations, and CRUD operations.

## Features

- MongoDB Atlas connection
- Mongoose schema and model creation
- Field validations
- Nested objects
- Array fields
- Default values
- Enum validation
- CRUD operations
- Timestamps support

## Technologies Used

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## Installation

Clone the repository:

```bash
git clone https://github.com/ifra489/mongodb-practice.git
```

Move into the project folder:

```bash
cd mongodb-practice
```

Install dependencies:

```bash
npm install
```

Create a `.env` file and add your MongoDB connection string:

```env
MONGO_URI=your_mongodb_connection_string
```

Run the project:

```bash
node index.js
```
